# Malcolm Dupuis ENC 1101 Portfolio

This is a static website portfolio for an ENC 1101 course.

## Files
- `index.html` — main website
- `style.css` — design and layout
- `assets/multimodal-literacy-narrative.pdf` — original essay PDF

## Launch on Vercel
1. Go to https://vercel.com/new
2. Choose "Import Third-Party Git Repository" or upload the folder to GitHub first.
3. Select this project folder/repository.
4. Framework preset: Other
5. Build command: leave blank
6. Output directory: leave blank
7. Click Deploy
